¿Cómo encontrar tu ID de usuario en eFootball?

En eFootball, tu ID de usuario se llama KONAMI ID / ID de juego. Lo puedes ver así:

  1. Entra al juego.
  2. Ve a Menú principal → Extras → Información de usuario / Perfil.
  3. Allí verás tu ID de usuario (un número largo). Ese es el que tienes que dar para que te agreguen como amigo.
